# App Functions & Benefits Guide

**A Complete Overview for Users**

---

## Introduction

Sleepless is South Africa's premier mobile application for discovering, booking, and experiencing events across all nine provinces. Whether you're looking for daytime festivals, pool parties, concerts, or late-night club events, Sleepless puts the entire entertainment landscape at your fingertips.

This guide provides a comprehensive overview of the app's features, functionality, and the benefits it delivers to event-goers and the entertainment community.

---

## Getting Started

### Account Creation

Creating a Sleepless account is quick and simple. Users can register using traditional email and password credentials, or take advantage of social login options for faster access.

| Registration Method | Process | Time Required |
|---------------------|---------|---------------|
| Email & Password | Enter username, email, password, and confirm | 2 minutes |
| Google Sign-In | One-tap authentication with Google account | 10 seconds |
| Apple Sign-In | Face ID/Touch ID authentication with Apple ID | 10 seconds |

Once registered, users gain immediate access to all app features including event discovery, bookings, and social features.

### Profile Customization

Sleepless allows users to personalize their experience through profile customization. Users can upload a custom avatar photo directly from their device gallery and set a display name that appears throughout the app. This personalization helps build community and makes social features more engaging.

## Core Features

### 1. Event Discovery

The heart of Sleepless is its powerful event discovery system. Users can explore events through multiple pathways designed for different discovery styles.

**Geographic Navigation**

Sleepless organizes events by location, making it easy to find what's happening in your area. The app covers all nine South African provinces:

| Province | Major Cities Covered |
|----------|---------------------|
| Gauteng | Johannesburg, Pretoria, Sandton |
| Western Cape | Cape Town, Stellenbosch |
| KwaZulu-Natal | Durban, Umhlanga |
| Eastern Cape | Port Elizabeth, East London |
| Free State | Bloemfontein |
| Mpumalanga | Nelspruit, Mbombela |
| Limpopo | Polokwane |
| North West | Rustenburg |
| Northern Cape | Kimberley |

Users navigate from province to city to view all events in their selected location.

**Smart Search**

The global search feature allows users to find events across all provinces instantly. Search works across multiple fields including event name, venue, city, artist, and event category. Recent searches are saved for quick access, and quick filter buttons help narrow results by event type.

**Calendar View**

For users who prefer planning by date, the calendar view displays events in a familiar monthly format. Days with scheduled events show a count badge, and tapping any date reveals the events happening that day. This feature is particularly useful for planning weekends or special occasions.

### 2. Event Information

Each event in Sleepless comes with comprehensive information to help users make informed decisions.

**Event Details Include:**

| Information | Description |
|-------------|-------------|
| Event Name & Poster | High-quality promotional artwork |
| Date & Time | When the event takes place |
| Venue & Location | Where the event is held with map access |
| Price | Ticket cost with any available tiers |
| Event Type | Category (Club, Festival, Concert, Pool Party, Rooftop) |
| Description | Full event details, lineup, and what to expect |
| Transport Options | Uber, taxi, and parking information |

**Location & Directions**

The location screen provides venue details and transport options. Users can view the venue address, access directions through their preferred navigation app, and see available transport methods including ride-sharing services, taxis, and parking availability.

### 3. Event Filtering

Finding the perfect event is easy with Sleepless's comprehensive filtering system.

| Filter Type | Options |
|-------------|---------|
| Event Type | Club Night, Festival, Concert, Pool Party, Rooftop |
| Price Range | Free, Under R100, R100-R300, R300+ |
| Date | Today, This Weekend, This Week, This Month |

Filters can be combined to narrow results precisely. For example, users can search for "Pool Parties under R200 this weekend" with just a few taps.

### 4. Ticket Booking

Sleepless provides a complete ticket purchasing experience without leaving the app.

**Booking Process:**

The booking flow is designed for simplicity and security. Users select their desired event, choose the number of tickets, review the total price including all fees, and proceed to payment. The entire process takes less than two minutes.

| Step | Action |
|------|--------|
| 1 | Select event and tap "Book Tickets" |
| 2 | Choose ticket quantity (1-10 tickets) |
| 3 | Review order summary and total price |
| 4 | Enter payment details |
| 5 | Confirm booking and receive QR ticket |

**Payment Methods:**

Sleepless supports multiple payment options to accommodate all users.

| Payment Method | Description |
|----------------|-------------|
| Credit/Debit Card | Visa, Mastercard, American Express |
| EFT | Direct bank transfer |
| Mobile Wallet | Digital payment solutions |

All transactions are secured with industry-standard encryption, and users receive instant confirmation upon successful payment.

### 5. QR Code Tickets

Every booking generates a unique QR code ticket that serves as the entry pass to the event.

**QR Ticket Features:**

| Feature | Benefit |
|---------|---------|
| Unique Verification Hash | Prevents counterfeiting and fraud |
| Offline Access | Tickets accessible without internet connection |
| Easy Scanning | Quick entry at venue gates |
| Digital Storage | No paper tickets to lose |

Users can access their tickets anytime from the "My Bookings" section. The QR code contains encrypted booking information that venues can verify instantly using the Sleepless scanner app.

### 6. Favorites & Bookmarks

The favorites feature allows users to save events they're interested in for later reference.

**How It Works:**

Users tap the heart icon on any event detail page to add it to their favorites. All saved events appear in a dedicated Favorites screen accessible from the home page. This feature is perfect for tracking events while deciding which to attend or comparing options before booking.

Favorites are stored locally on the device and persist across app sessions, ensuring users never lose track of events they've marked.

### 7. Event Reminders

Never miss an event with Sleepless's reminder system. Users can set push notification reminders for any event at customizable intervals.

| Reminder Option | When Notification Arrives |
|-----------------|---------------------------|
| 1 Hour Before | 60 minutes before event start |
| 1 Day Before | 24 hours before event start |
| 1 Week Before | 7 days before event start |

Reminders appear as push notifications on the user's device, even when the app is closed. This ensures users have time to prepare, arrange transport, and arrive on time.

### 8. Social Features

Sleepless includes social features that enhance the event experience.

**Event Sharing**

Users can share event details with friends through any installed app including WhatsApp, SMS, email, Instagram, and more. The share function generates a formatted message with event name, date, venue, and a link to view the event in Sleepless.

**Comments & Community**

Each event has a comments section where users can discuss the event, share excitement, ask questions, and connect with others planning to attend. Users can like comments and engage with the community.

### 9. Push Notifications

Stay informed with Sleepless push notifications. Users can enable notifications in the About section to receive updates about new events in their area, special promotions, and event reminders.

| Notification Type | Content |
|-------------------|---------|
| New Events | Alerts when events are added in favorite cities |
| Reminders | Pre-event notifications for saved events |
| Promotions | Special offers and early-bird discounts |
| Updates | Changes to booked events (time, venue, etc.) |

Notifications can be enabled or disabled at any time through the app settings.

## Navigation & User Experience

### Intuitive Navigation

Sleepless is designed for effortless navigation with multiple ways to move through the app.

| Navigation Method | How It Works |
|-------------------|--------------|
| Back Button | Tap the arrow icon to return to the previous screen |
| Home Button | Tap the home icon to return to the main screen |
| Swipe Gesture | Swipe from the left edge to go back (iOS/Android) |
| Breadcrumb Trail | Tap any level in the breadcrumb to jump directly |

The breadcrumb trail shows the current navigation path (e.g., "Home > Gauteng > Joburg > Event Name"), allowing users to jump to any level without going back step by step.

### Smooth Animations

All screen transitions feature smooth slide animations that provide visual feedback and create a polished, premium feel. Screens slide in from the right when navigating forward and slide out to the right when going back.

### Frequently Asked Questions

**How do I get a refund?**
Contact the event organizer directly through the app's event page. Refund policies vary by event.

**My QR code won't scan?**
Ensure screen brightness is at maximum and the code is displayed clearly. If issues persist, show your booking confirmation email as backup.

**How do I change my password?**
Go to Profile > Settings > Change Password. You'll need to enter your current password to make changes.

**Can I transfer tickets to someone else?**
Ticket transfers depend on the event organizer's policy. Contact them through the event page for transfer requests.
