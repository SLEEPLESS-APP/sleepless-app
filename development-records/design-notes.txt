## Design Style
- Dark/blurred background with bokeh lights effect
- Gradient colors (purple, blue, orange, warm tones)
- White text on dark backgrounds
- Rounded input fields with icons
- Circular profile avatars and event images
- Semi-transparent UI elements
- Cursive/script font for "Sleepless" logo

## Color Palette (estimated)
- Primary background: Dark purple/blue gradient
- Accent: Orange/coral for buttons
- Text: White
- Input backgrounds: Semi-transparent white/gray
- Icons: White with circular borders
